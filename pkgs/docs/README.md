# メモをまとめるマークダウン

## 必要な機能

- フロントエンド
    - Lit
    - Biconomy

- スマートコントラクト
    - NFT(ERC1155)
    - Gameコントラクト
        - Game名
        - GameID
        - 早押しの回数
        - winnerのアドレス
        - 参加者のアドレスを格納する配列
        - ゲーム実施中であるか否かのフラグ
        - winnerへの賞金送金機能
        - 参加者へのNFTMint機能

- スクリプト
    - メタデータをアップするためのスクリプト
