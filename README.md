# WakuWakuOnchainGame
WakuWakuOnchainGame用のリポジトリです。

## 動かし方

- インストール

  ```bash
  yarn
  ```

- フロントの起動

  ```bash
  yarn frontend:dev
  ```

- メタデータ類のアップロード

  ```bash
  yarn upload
  ```
